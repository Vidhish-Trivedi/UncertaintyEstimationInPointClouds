### Repository Structure

Starting at the repository's root:

| Path                       | Description                                                                       |
| -------------------------- | --------------------------------------------------------------------------------- |
| Code/open3d.ipynb          | Experiment to visualize point cloud data                                          |
| Code/vaihingen-area2.ipynb | Random Forest Classifier on Vaihingen Dataset                                     |
| Code/features.ipynb        | Code to compute geometric features for point cloud data                           |
| Code/features_list1.txt    | Computed geometric features for 1st half of Vaihingen dataset - "area2_cov_multi" |
| Code/features_list2.txt    | Computed geometric features for 2nd half of Vaihingen dataset - "area2_cov_multi" |
| Code/pyspark/demo.ipynb    | A demo ML task to get familiar with PySpark                                       |
| Code/NN/NN.ipynb           | Implementation of Gaussian Discriminant Analysis                                  |
| Code/NN/gda.py             | Trying a simple Neural Network Classifier for point-cloud data                    |
